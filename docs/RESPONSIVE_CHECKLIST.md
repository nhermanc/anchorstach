# Responsive layout checklist (manual)

After header / home section updates, verify in Chrome DevTools (Toggle device toolbar) or resize the window:

| Viewport width | What to check |
|----------------|---------------|
| **375** | Hamburger + drawer; logo readable; hero stacks; no horizontal scroll |
| **480** | Same; header CTA hidden — use drawer **GET STARTED** |
| **568** | Same |
| **768** | Tablet: hamburger, hero carousel stacked, work grid 2-ish columns |
| **900** | Hamburger; testimonial + work centered |
| **1024** | Same (still &lt; 1200 → menu icon) |
| **1199** | Last width before full nav — no overlap, CTA visible |
| **1200** | Full horizontal nav, no hamburger; hero side‑by‑side |
| **1366** | Laptop default — header + hero + grids comfortable |
| **1500+** | Large desktop — max-width gutters even |

Also test **orientation** on a real phone/tablet if possible.
